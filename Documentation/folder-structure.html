Folder Structure

Let's go through the files and folders of the app.

In the root of the project we have the following important files and folders.

1. /node_modules

The npm packages installed in the project with the <npm install> command.

2. /src

This folder is the place where the developers do most of their work. When you navigate into the Src folder you’ll be able to see 4 other folders inside the src folder. They are app folder, assets folder, pages folder, theme folder.

Under this folder you will find the following important folders and files.

3. /assets

Assets Folder contains all the images, fonts which are used in the app. Also other assets of the app goes here too.

4. /environments

Under this folder are configuration files used by the Angular CLI to manage the different environment variables.

6. /src/index.html

src/index.html is the main entry point for the app, though its purpose is to set up scripts, CSS includes, and bootstrap, or start running our app. We won’t spend much of our time in this file.

Let’s start with the app folder

The App folder is the largest folder because it contains all the code of our ionic app. It has all the components, modules, pages, services and styles you will use to build your app.

Inside the /app folder there are also some important files:

app.component.html

This serves as the skeleton of the app. Typically has a to render the routes and their content. It can also be wrapped with content that you want to be in every page. In this app we added the side menu in this file.

app.component.ts

It defines the basic structure and initial navigation of the app.

app-routing.module.ts

Here we define the main routes. Child routes of other lazy modules are defined inside those modules. These routes are registered to the Angular RouterModule in the AppModule.

app.module.ts

This is the main module of the ionic project.

Component Folder

This folder contains the components which are used many pages all over the apps.

Pages Folder

Each page has its own folder. Within that folder you will find every related file for that page. This includes the html for the layout, sass for the styles and the main page component.

Service Folder

This folder is for all the services you will use to access the data that will be presented in the app. In this first part we will only focus on the local data json files.
