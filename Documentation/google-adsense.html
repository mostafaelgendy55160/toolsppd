Google Adsense

For easy use google adsense we are using `ng2-adsense` plugin for this Angular Applications. Its easy to use for responsive ads, ads with specified width and same component but w/ setTimeout when the params are changed to refresh the ads.

Github

Configure Google Adsense

Google AdSense is a simple and free tool that will allow you to earn money by displaying ads on your site.

You just enter your Google Adsense Client and Google Adsense Slot in the project modules. Go into the project src/environments/environment.ts`
src/environments/environment.prod.ts`
modules and replace with your Adsense client and slot number. 

adClient: 'ca-pub-xxxxxxxxxxxxxxxx', // Google Adsense AdClient
adSlot: xxxxxxxxxxxxxx, // Google Adsense AdSlot

We used the ng2-adsense plugin for integration google adsense, see the documentation from here- https://github.com/scttcper/ng2-adsense

Dependencies(ng2-adsense)

Easy AdSense for Angular Applications

Install

npm install ng2-adsense

Import NgModule

Add AdsenseModule to the imports of your NgModule

import { AdsenseModule } from 'ng2-adsense';

@NgModule({
    imports: [
    // shown passing global defaults (optional)
    AdsenseModule.forRoot({
        adClient: 'ca-pub-7640562161899788',
        adSlot: 7259870550,
    }),
    ...

Show Ad

Uses global defaults which can be overriden via inputs

< ng-adsense
        [adClient]="'ca-pub-7640562161899788'"
        [adSlot]="7259870550"
        [display]="'inline-block'"
        [width]="320"
        [height]="108"
      >
      < / ng-adsense >
